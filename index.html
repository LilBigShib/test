Mozilla Firefox

Mozilla Firefox — свободный браузер на движке Quantum, разработкой и распространением которого занимается Mozilla Corporation. Четвёртый по популярности браузер в мире и первый среди свободного ПО. В России Firefox занимает четвёртое место по популярности среди браузеров для ПК.

Браузер Mozilla Firefox имеет ряд преимуществ

  • Безопасность
  • Лёгкая архитектура
  • Наличие мастер-пароля
  • Удобная работа с закладками в этой программе
  • Блокировка всплывающих окон
  • Синхронизация настроек и данных на разных компьютерах

Установить браузер можно по этой ссылке Firefox Browser
